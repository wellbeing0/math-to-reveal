# Support

Math Rewards is a small open source educational game. Support is handled through GitHub issues.

## Questions And Bugs

Use a GitHub issue for reproducible bugs, setup trouble, browser/device problems, or questions about contributing.

Before filing a bug, include:

- the device and browser;
- steps to reproduce;
- what happened;
- what you expected instead.

## Security

Do not report security issues in public issues. Follow `SECURITY.md` for private reporting.

## Private Family Deployments

This public repo does not include private family deployment support, private media packs, generated voice files, or host-specific operations notes.
