# Changelog

## 0.3.0 - 2026-05-29

- Add optional workbench handoff support for deployments that provide a compatible manipulative workbench URL.
- Tighten prompt instruction sizing for long fraction and decimal prompts across tablet and phone layouts.
- Keep teaching aids support-only and reward-neutral when a prompt is completed after repeated misses.
- Add icon-labeled command buttons across the child game, adult settings, teaching aid panels, and optional workbench control.
- Hide prompt-specific teaching help until the child chooses Think or Clue.
- Add phone regression coverage for icon command rows, one-line controls, and horizontal overflow.

## 0.2.0 - 2026-05-28

- Prepare Math Rewards for a sanitized public open source release.
- Add K-4 math lanes, video reveal rewards, repeat-instruction audio fallback, adult grade/operation settings, and local progress.
- Add answer-cycling support, teaching aids, prompt-aware teaching visuals, and optional GIPHY reward themes.
- Keep private assets and deployment operations out of the public release surface.
