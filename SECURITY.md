# Security Policy

## Supported Versions

Security reports should target the current public release line.

## Reporting

Open a private security advisory in GitHub when available. If that is not available, contact the repository owner privately.

Do not file public issues for secrets, bypasses, private deployment details, or data exposure concerns until the issue has been reviewed.

## Scope

Math to Reveal is a static browser app. It should not require runtime secrets, paid APIs, telemetry, accounts, or cloud sync for public gameplay.
