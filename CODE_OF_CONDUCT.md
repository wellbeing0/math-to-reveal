# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

## Standard

Participate respectfully, keep discussion focused on the work, and make the project approachable for people with different backgrounds and experience levels.

Unacceptable behavior includes harassment, personal attacks, discriminatory language, publishing private information, or deliberately disrupting the project.

## Enforcement

Report concerns privately to the repository owner. Maintainers may remove comments, close issues, reject contributions, or restrict participation when needed to keep the project healthy.
