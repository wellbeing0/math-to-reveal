# Asset Policy

Math Rewards source is MIT licensed. Media and generated audio can have different terms and must be documented separately.

## Public Release Assets

- Public source release may include one Pexels reward video sample and matching poster.
- The sample must retain attribution and Pexels license notes near the media files.
- Public gameplay must still work if reward media is replaced by local user-provided files.
- Public builds must not require Gemini WAVs or paid API access.

## Private Family Assets

- The current full Pexels kitten reward set is treated as private family build content unless explicitly exported.
- Gemini Aoede instruction WAVs are private generated assets.
- Family photos, videos, voice clips, generated manifests, protected deployment scripts, and host-specific notes belong in private/ or another ignored private overlay.

## Contributor Assets

Contributors adding assets must include:

- source URL or creation notes;
- license name and link;
- attribution text when required;
- confirmation that the asset is appropriate for an elementary-age educational game.

Do not add assets with unclear provenance, non-commercial-only terms, or terms that prevent redistribution in a public repository.
